# Strutture di controllo
Sono le stesse di C.

Esistono versione compatte degli if:
```java
numero >= 0 ? numero : -numero;

// {proposizione} ? {se vero} : {se falso};
```

#### Potenze
```1E10``` equivale a $ 10^{10} $.

#### Switch

```java
switch({espressione byte, short, int o char}){
    case {caso1} : istruzione; break;
    [etc...]
}
```

Si possono usare anche gli enum negli switch.

##### NB conviene convertire certe stringhe in maiuscolo usando ```toUppercase()```.
